# KiCad library collection

This repository contains my custom KiCad libraries for personal use in board-level projects.

## Library Structure
### Power_Measurement
Electrical power and energy measurements.
#### SEN0291
Description: I2C Digital Wattmeter Module  
Manufacturer: DFRobot  
Documentation: [DFRobot Wiki](https://wiki.dfrobot.com/Gravity:%20I2C%20Digital%20Wattmeter%20SKU:%20SEN0291)

### Power_Conversion
Level conversion of electrical power.
#### DFR1015
Description: Buck Converter Module  
Manufacturer: DFRobot  
Documentation: [DFRobot Store](https://www.dfrobot.com/product-2599.html)

### Motor_Control
Motor control.
#### DRI0054
Description: Raspberry Pi Motor Driving Board  
Manufacturer: DFRobot  
Documentation: [DFRobot Wiki](https://wiki.dfrobot.com/sku_dri0054_raspberry_pi_driving_board)
